1) Запуск ПО через папка Solution/Solution_Support_GIT#4

2) При запуске создаются TCP сервера и сервер Client, происходит подключение клиента к серверам.
	//выбирая один из вариантов, отправляет серверам TCP  сообщение "1", "2", "0"

	"0 - exit, 1 - Read, 2 - NoRead: "

	При получении сообщения от клиента, TCP сервера создают ответное сообщение Клиенту. При сообщении Клиента "1" - генерируем сообщения с текущей датой - "дата1" и датой конца - "дата2"
	При сообщении Клиента "2" - генерируем три рандомных сообщение, одно сообщения как с вариантом "1" и два других сообщения в котоорых одна из дат 1 или 2 равны "000000".
	Одно из этих сообщений будет выдаваться каждым TCP сервером рандомно, при выборе n раз "2" можно увидет вариации как приложение и Клиент_Сервер отрабатывают событие "NоRead".
